Paso 1. El usuario debe cargar la base de datos para el meta-análisis en el menú de Cargar datos . Unos datos de ejemplo están disponibles y cualquier otra base de datos puede subirse de un fichero externo en formato *.csv separado por ';'.

Paso 2. Una vez subidos los datos, el usuario debe escoger entre un análisis cuantitativo o cualitativo.

Paso 3. Seleccionar las variables correspondientes en orden dependiendo del análisis escogido.

Para el Análisis Cuantitativo, el usuario debe seleccionar las cinco variables en el siguiente orden:

  • Variable 1: Nombre Compuesto.
  • Variable 2: P-Valor.
  • Variable 3: Cambios en la concentración/intensidad.
  • Variable 4: N total.
  • Variable 5: Referencias.

Para el Análisis Cualitativo, el usuario debe seleccionar las tres variables en el siguiente orden:

  • Variable 1: Nombre Compuesto.
  • Variable 2: Comportamiento.
  • Variable 3: Referencias.

Paso 4. Una vez seleccionadas las variables adecuadas, el botón de Calcular aparecerá para empezar con el análisis. El usuario debe clicarlo para obtener los resultados que se visualizarán automáticamente en las pestañas correspondientes:

  • Análisis Cuantitativo muestra els resultados de la aplicación del paquete amanida, un meta-análisis con pesos basado en resultados generales. Combina p-valores mediante un método de Fisher con pesos y los cambios en la concentración/intensidad mediante una media ponderada, donde el peso es el tamaño de estudio (n). Los resultados se presentan en una tabla y gráficamente con una gráfica Volcano.
  • Análisis Cualitativo muestra el análisis exploratorio de los datos. La gráfica de votos es el resultado del análisis de conteo de votos, los votos están recogidos de la siguiente manera: +1 para los regulados al alza, -1 para los regulados a la baja y 0 para los que no cambian. El gráfico exploratorio es una variación de la gráfica de votos donde los compuestos visualizados estan dividos por tendencia (a la alza/a la baja) para encontrar discrepancias en el resultado.
  • Informe contiene la descarga del informe del análisis especificado.

Si el análisis es cuantitativo, tanto resultados cuantitativos como cualitativos seran mostrados. Para el análiais cualitativo, solo resultados cualitativos serán mostrados.

Todos los análisis se han hecho con el paquete de R Amanida. Más información de estos análisis se puede encontrar en: Amanida Guía de Usuario .
