Step 1. The user must load the data set for the meta-analysis in the Load data menu. An example data set is available, and any other user’s data set can be uploaded from an external file in *.csv format separated by ';'.

Step 2. When uploaded the data, the user must choose between a quantitative or a qualitative analysis.

Step 3. Choose corresponding variables in order depending on the analysis chosen.

For Quantitative Analysis, the user’s must select five variables in the following mandatory order:

  • Variable 1: Compound Name.
  • Variable 2: P-Value.
  • Variable 3: Fold-change.
  • Variable 4: N total.
  • Variable 5: References.

For Qualitative Analysis, the user’s must select three variables in the following mandatory order:

  • Variable 1: Compound Name.
  • Variable 2: Behaviour.
  • Variable 3: References.

Step 4. Once the variables are selected accordingly, the Calculate button will be displayed to begin with the analysis. The user must click on it to obtain the results that will be automatically displayed in the corresponding tabs:

  • Quantitative Analysis shows the results of applying amanida, a weighted meta-analysis based on overall results. It combines p-values by a weighted Fisher method and fold-change by weighted average, where the weight factor is the study size (n). Results are disclosed in a table and graphically through a volcano plot.
  • Qualitative Analysis shows the exploratory analysis of data. Vote plot is the results of a vote-counting analysis, votes are computed as followed +1 for up-regulation, -1 for down-regulation, and 0 for no change. The explore plot, is a variation of the vote-counting plot where compounds plotted are divided by trend (up-regulated/down-regulated) to find discrepancies in results.
  • Report will have available the downloading of the report of the analysis specified.

If the analysis mode is quantitative, both quantitative and qualitative results will be displayed. For qualitative analysis, only qualitative results will be showed.

All analysis are done with R package Amanida. More information on these analysis can be found in the following vignette: Amanida User Guide .
